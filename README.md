# Hybrid Serializable Closure

Hybrid Core Serializable Closure provides an easy and secure way to serialize closures in PHP.

## Requirements

* WordPress 6.1.0+.
* PHP 8.1+.
* [Composer](https://getcomposer.org/) for managing PHP dependencies.
